## <a id="fr-installation">Installation</a>

### <a id="fr-win">Windows</a>

1. Télécharger le dossier <a href="https://github.com/Hypocrate-code/HypLoad/releases/download/v.1.0.0/windows" target="blank">.zip</a> de la dernière version.
2. Dézipper le dossier.
3. Exécuter le fichier <u>Hypload-Setup.exe</u>
4. Servez vous de HypLoad !

Les téléchargements se retrouveront dans les dossiers *NomUtilisateur/Music/HypLoad/* et *NomUtilisateur/Vidéos/HypLoad/* respectivement. Voir [Fonctionnalités](#)

## <a id="fr-features">Fonctionnalités d'HypLoad</a>

### <a id="fr-download-video">Télécharger une vidéo</a>

1. Copiez le lien de votre vidéo YouTube (Partager + Copier)
2. Cliquez sur "Télécharger une vidéo" sur HypLoad et collez votre lien.
3. Vérifiez que l'option de téléchargement vous convient : "Audio uniquement" pour télécharger un mp3, ou "Audio et Vidéo" pour un mp4, sinon cliquez dessus.
4. Cliquez sur Télécharger.

Les téléchargements se retrouveront, s'il n'y a pas eu d'erreur, dans [ces dossiers](#fr-folders).


### <a id="fr-download-playlist">Télécharger une playlist</a>

Vous pouvez lancer le téléchargement d'une playlist entière :
1. Copiez le lien de votre playlist YouTube (Partager + Copier)
2. Cliquez sur "Télécharger une playlist" sur HypLoad et collez votre lien.
3. Vérifiez que l'option de téléchargement vous convient : "Audio uniquement" pour télécharger un mp3, ou "Audio et Vidéo" pour un mp4, sinon cliquez dessus.
4. Cliquez sur Valider.

Une fois la playlist chargée, chaque vidéo a une option Audio uniquement LOGO ou Audio et Vidéo LOGO ; ainsi qu'une case cochée LOGO ou non LOGO, indiquant si la vidéo doit être téléchargée. Pour inverser l'option, cliquez dessus.

Si la vidéo est déjà présente dans son [dossier correspondant](#fr-folders), sa case sera par défaut décochée pour vous éviter de la télécharger de nouveau.

HypLoad enregistre les liens de vos playlists dans la section [Charger vos playlists](#)

## <a id="fr-folders">Dossiers de téléchargement</a>

### <a id="fr-win-folders">Windows</a>

- Option Audio uniquement &rarr; dans le dossier C:/Users/votreNom/Music/HypLoad/, soit **Win + R + "Music", dossier "HypLoad"**
- Option Audio et Vidéo &rarr; dans le dossier C:/Users/thiba/Videos/HypLoad, soit **Win + R + "Videos", dossier "HypLoad"**

### <a id="fr-mac-folders">macOS</a>

- Option Audio uniquement &rarr; dans le dossier nomUtilisateur/Musique/HypLoad/, soit **Finder + Musique/HypLoad**
- Audio et Vidéo &rarr; dans le dossier nomUtilisateur/Vidéos/HypLoad/, soit **Finder x+ Vidéos/HypLoad**
